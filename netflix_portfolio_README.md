# Ranjitha Rani — Portfolio

**SWE in Making** · Netflix-style portfolio site

A standalone, single-file portfolio built to look and feel like a Netflix original. No frameworks, no build step, no dependencies — just drop `index.html` into any static host and it works.

## Deploy to GitHub Pages

1. Create a new repo (e.g. `ranjitharani-netflix-portfolio`)
2. Push this folder to `main`
3. Go to **Settings → Pages → Source → main / root**
4. Your site will be live at `https://<your-username>.github.io/ranjitharani-netflix-portfolio/`
