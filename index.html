About

behind the stack

I'm a Master's student in Computer Science at the University of Oregon, originally from Bengaluru, India. I completed my B.Tech at PES University before crossing the world to work on problems that don't have textbook answers.

My thesis is building SAM CFT — a novel crash fault tolerance protocol for disaggregated memory using RDMA. It sits at the intersection of distributed systems, hardware architecture, and consensus theory. That's the kind of intersection I like.

I work across the full stack — from low-level C/C++ memory protocols to cloud-native Kubernetes infrastructure to React frontends. JavaScript, TypeScript, Java, Go, Python — whatever the system needs.
